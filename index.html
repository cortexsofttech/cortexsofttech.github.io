CtrlX

A wireless Bluetooth HID controller built on ESP32 with an OLED display. Five input modes — Mouse, Presentation, Gaming, Media, and System control — with BLE 4.2 connectivity and smart deep sleep.

  • 5 switchable modes via joystick & buttons
  • BLE HID — Keyboard + Mouse combo
  • SSD1306 128×64 OLED over I²C
  • 3-min idle → auto deep sleep
  • Battery monitoring via ADC
